# Project Description

This project includes various data-analytics skills including 
batch extraction with `sql-connector` modules for databricks and solving
business problems with analysis upon extracted data.

`business_questions` : a directory containing business problems
generated by GPT for me to solve

`challenge_deliverables` : a directory containing deliverable artifacts
based on business questions

`collection` : the collection of queries used to complete data
analysis tasks

`data` : a directory containing extracted data files in various 
formats

`src` : the main directory where `extract` functions and main
executables are kept. Will be diversified as business problems
are collected.
